# Racines+ — Arbre généalogique numérique

Projet d'arbre généalogique numérique **à grande échelle** : quartier → village → région → pays → diaspora. Représentation **pyramidale**, collaborative, avec validation des données et export.

**Slogan** : *« Racines+, l'Histoire continue avec toi. »*

---

## Travail à la racine — Version web

- **Dossier du projet** : `C:\Racines+` (travail à la racine, pas de sous-dossier dédié).
- **Version cible** : **web** en premier (application + API).
- **Règles** : code propre, sécurisé, pas de secrets dans le code ; conformité RGPD et bonnes pratiques (sécurité, UI/UX, qualité des données).
